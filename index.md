Introduction

Data Structure is a way of storing and organizing data so that it is easy to access, modify and Analyze. An example of the data structure is in telephone directory names are sorted in alphabetical order so as to make it easy for a person to search for a particular name in the telephone directory without sort of data structure it will be highly impossible to find a particular name in the telephone directory. Another example is a balance sheet in which the debit and credit is mentioned separately, such way of organizing the data makes it easier to understand the overall financial situation in a single glance of the information.
